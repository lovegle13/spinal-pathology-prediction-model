# spinal-pathology-prediction-model
Данный проект представляет собой специализированный анализ и прогнозирование динамики заболеваний позвоночника пациента на основе временных рядов (2019-2026 гг.).
Анализ охватывает следующие патологии:
Грыжи: шейный отдел (C4/C5, C5/C6), грудной отдел (Th7/Th8), поясничный отдел (L4/L5, L5/S1)
Стеноз позвоночного канала (C4/C5, C5/C6)
Протрузии (C3/C4, C6/C7, Th8-Th12)
Ретролистез (L4)
Гемангиолипомы (Th6,Th12, L2-L5)

Также включены данные о:
Эффективности лечения (блокады, РЧА, PRP-терапия)
Динамике болевого синдрома (по шкале ВАШ)
Развитии миелопатии
Клинических событиях

Методология
Сбор и структурирование данных из различных источников за период 2019-2026
Визуализация динамики состояния позвоночника с помощью matplotlib/seaborn
Машинное обучение с использованием Gradient Boosting для прогнозирования:
Размеров грыж C4/C5 и C5/C6
Степени стеноза

Используемые технологии
Python 3
Pandas, NumPy
Matplotlib, Seaborn
Scikit-learn (GradientBoostingRegressor)
XGBoost

Анализ эффективности различных методов лечения
Прогнозируется стабилизация размеров грыж на уровне 0.55 см (C4/C5) и 0.45 см (C5/C6)
Выявлена фармакорезистентность к блокадам в 2026 году
Обнаружено развитие начальной миелопатии, требующее нейрохирургического вмешательства
РЧА (радиочастотная абляция) показала временную эффективность (снижение боли на 50-70%)

Рекомендации
На основе анализа рекомендовано:
Срочная консультация нейрохирурга (миелопатия)
Рассмотрение оперативного лечения
Контрольная МРТ каждые 6 месяцев
Поиск альтернатив блокадам
