# spinal-pathology-prediction-model 
СПЕЦИАЛИЗИРОВАННЫЙ АНАЛИЗ И ПРОГНОЗ ПО СОСТОЯНИЮ ПОЗВОНОЧНИКА + ИТОГОВЫЙ АНАЛИЗ МНЕНИЙ И ЛЕЧЕНИЯ

Данный проект представляет собой комплексный анализ клинических данных пациентки с множественными грыжами шейного отдела, стенозом позвоночного канала и центральной сенситизацией.

Проект объединяет:
Анализ динамики грыж и стеноза за 7 лет (2019–2026)
Прогнозирование дальнейшего развития заболевания на 2027–2028 гг.
Систематизацию и визуализацию мнений 6 специалистов по вопросу оперативного лечения
Оценку эффективности проведенной терапии
 
Анализ охватывает следующие патологии:
Грыжи: шейный отдел (C4/C5, C5/C6), грудной отдел (Th7/Th8), поясничный отдел (L4/L5, L5/S1)
Стеноз позвоночного канала (C4/C5, C5/C6)
Протрузии (C3/C4, C6/C7, Th8-Th12)
Ретролистез (L4)
Гемангиолипомы (Th6,Th12, L2-L5)

Также включены данные о:
Эффективности лечения (блокады, РЧА, PRP-терапия)
Динамике болевого синдрома (по шкале ВАШ)
Развитии миелопатии?
Клинических событиях

Методология:
Сбор и структурирование данных из различных источников за период 2019-2026
Визуализация динамики состояния позвоночника с помощью matplotlib/seaborn
statsmodels — модель Хольта для прогнозирования стеноза (временные ряды):
Размеров грыж C4/C5 и C5/C6
Степени стеноза

 Для прогноза стеноза применена модель Хольта (двойное экспоненциальное сглаживание), которая учитывает тренд и работает с малым количеством наблюдений.
Построены два ручных сценария:
консервативный: стеноз остается на текущем уровне (0,7 см).
прогрессивный: медленное ухудшение на 0,01 см/год.

Используемые технологии:
Python 3.8+
pandas, numpy — обработка данных
matplotlib, seaborn — визуализация
statsmodels — модель Хольта (прогнозирование временных рядов)
Google Colab / Jupyter Notebook — среда выполнения

Анализ эффективности различных методов лечения:
Прогнозируется стабилизация размеров грыж на уровне 0.60 см (C4/C5) и 0.50 см (C5/C6).
Выявлена фармакорезистентность к блокадам в 2026 году.
Обнаружено развитие начальной миелопатии ?, требующее нейрохирургического вмешательства.
РЧА (радиочастотная абляция) показала временную эффективность (снижение боли на 50-70%).

Рекомендации
На основе анализа рекомендовано:
Срочная консультация нейрохирурга (миелопатия?)
Операция — только при прогрессировании неврологического дефицита (мнения разделились: 4 против, 2 за)
МРТ — каждые 6–12 месяцев
Отказ от блокад (фармакорезистентность + рост "сахара")
Альтернативное обезболивание (лечение центральной сенситизации)
Эргономика, исключение статики
Коррекция дефицитов, нейропротекция, лечение тревоги

Несмотря на детальный анализ данных и консультации ведущих специалистов, окончательная ясность в выборе тактики лечения не достигнута — единого мнения нет, решение остаётся за пациентом.
